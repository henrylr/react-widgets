In addition to the main package, there are a few React widget compatible addons.

## `react-widgets-tailwind`

If you are using TailwindCSS for your web app and want to integrate react-widgets with your theme
`react-widgets-tailwind` provides a tailwind plugin as an alternative to the core CSS/Sass styles.

## `react-widgets-vritualized`

A set of helpers for adding virtualization or "windowing" to widget dropdowns for
handling very large lists. Built on `react-list`.
